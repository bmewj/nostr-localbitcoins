# Nostr LocalBitcoins - <small>[LNbits](https://github.com/lnbits/lnbits) extension</small>
<small>For more about LNBits extension check [this tutorial](https://github.com/lnbits/lnbits/wiki/LNbits-Extensions)</small>

## Inspired by LocalBitcoins: a P2P currency exchange marketplace built on Nostr

### Usage

1. Enable extension
2. Create a TPOS\
   ![create](https://imgur.com/8jNj8Zq.jpg)
3. Open TPOS on the browser\
   ![open](https://imgur.com/LZuoWzb.jpg)
4. Present invoice QR to customer\
   ![pay](https://imgur.com/tOwxn77.jpg)
